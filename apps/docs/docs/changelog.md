# Changelog

All notable changes to Raven Docs.

## v0.10.0 <span className="version-badge">Latest</span>

*January 2026*

### New Features

- **MCP Tool Search** - Agents can now search for relevant tools instead of loading the entire catalog
- **Slack Integration** - Get notifications and search Raven Docs from Slack
- **Discord Integration** - Team notifications in Discord channels

### Improvements

- Improved database connection handling for Cloud SQL
- Better error messages for API validation errors
- Faster page load times with optimized queries

### Bug Fixes

- Fixed workspace creation error with missing columns
- Fixed static file serving in production Docker image
- Resolved migration script issues in production environment
